ЛІТАЙ, МРІЙ, ВИВЧАЙ!

Мультимедійне електронне навчально-розважальне видання для дітей молодшого шкільного віку

Щоб ознайомитися з книжкою виконайте наступні кроки:

Для Android:

  1. Завантажте файл;
  2. Перейдіть за посиланням нижче і встановіть програму для читання електронних книжок;
  3. Відкрийте програму, натисніть на "+" в правій нижній частині екрана;
  4. Оберіть завантажений файл.

Для IOS (Apple):

  1. Завантажте файл;
  2. Відкрийте завантажений файл за допомогою Apple Books.
